Dieses Plugin erweitert Shopware's HTTP Cache um einen IP Filter. Zugriffen durch entsprechende Clients werden keine pre-cached Seiten ausgegeben. Weiterhin kann der Cache durch zuvor festgelegte URL Parameter und Cookies, sofern diese gesetzt sind, umgangen werden.

Nach der Installation sollte die config.php im Shopware Stammverzeichnis in etwa so aussehen:

    require_once __DIR__ . '/custom/plugins/FroshHttpCacheIpExclude/Components/IpExcludeStore.php';
    return array (
      'db' =>
      array (
        // ...
      ),
      // ...
      'httpcache' =>
      array (
        'storeClass' => 'FroshHttpCacheIpExclude\\Components\\IpExcludeStore',
        'extended' =>
        array (
          'passedStoreClass' => NULL,
          'ipExcludes' =>
          array (
          ),
          'paramExcludes' =>
          array (
          ),
          'cookieExcludes' =>
          array (
          ),
        ),
      ),
    );

Das Element ipExcludes unter extended enthält ein Array an IP Adressen, denen in der Folge kein Cache mehr ausgespielt wird.

Das Element passedStoreClass, ebenfalls unter extended kann dazu genutzt werden einen weiteren Cache Store, bspw für Redis Cache, anzugeben.

Weiterhin kann das Element paramExcludes bzw cookieExcludes dazu genutzt werden, Namen von URL Parametern, respektive Cookies, in einem Array anzugeben. Sind die enthaltenen Parameter/Cookies gesetzt, wird der Cache ebenfalls umgangen.
